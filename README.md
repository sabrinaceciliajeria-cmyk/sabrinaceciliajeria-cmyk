## Proyectos Destacados

### ✦ PetVision

Sistema de gestión veterinaria full stack desarrollado con arquitectura separada en frontend y backend, enfocado en la administración clínica, organización de pacientes (mascotas), reservas de atención y mejora de la experiencia de usuario en entornos veterinarios digitales.

El proyecto implementa una arquitectura modular donde el backend gestiona la lógica de negocio, persistencia de datos y APIs, mientras que el frontend se encarga de la interfaz de usuario, consumo de servicios y experiencia visual interactiva.

### ✦ Pokédex Retro

Dashboard interactivo de Pokémon desarrollado con JavaScript Vanilla que consume datos en tiempo real desde la API pública PokéAPI.

El proyecto evolucionó desde una Pokédex retro clásica hacia una aplicación moderna enfocada en experiencia de usuario, renderizado dinámico y diseño responsive.

Incluye funcionalidades como búsqueda dinámica de Pokémon, visualización de estadísticas, sistema de favoritos con persistencia en localStorage, navegación entre Pokémon, soporte para Dark / Light mode y consumo completo de datos desde una API REST.
